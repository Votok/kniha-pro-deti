# Kniha pro děti

Web pro dětské knihy od Pavlíny Jurkové a Jarmily Vlčkové.

## Tech Stack

- [Eleventy](https://www.11ty.dev/) (Static Site Generator)
- Bootstrap 5
- Vanilla JS

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

## Build

```bash
npm run build
```
